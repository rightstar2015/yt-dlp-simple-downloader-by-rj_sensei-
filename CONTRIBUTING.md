## 開發指南

### 環境設置
1. 克隆倉庫
2. 建立虛擬環境：`python -m venv .venv`
3. 啟動虛擬環境
4. 安裝依賴：`pip install -r requirements.txt`

### 代碼規範
- 使用 Python 3.8+ 語法
- 遵循 PEP 8 風格指南
- 為新功能添加註釋

### 提交變更
- 確保代碼無語法錯誤
- 在本地測試所有功能
