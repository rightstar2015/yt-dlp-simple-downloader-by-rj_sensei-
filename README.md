# yt-dlp Simple Downloader by Odudins

一款簡單好用的 yt-dlp 圖形化下載器，基於 Python 和 PyQt5 開發。

## 功能特性
-  **簡單模式** - 一鍵下載，支援 MP4、MP3、M4A 格式
-  **進階模式** - 自由選擇影片和音頻格式組合
-  **片段下載** - 按指定時間戳下載影片片段
-  **自動更新** - 一鍵從 GitHub 下載最新的 yt-dlp
-  **進度顯示** - 實時顯示下載進度和預估剩餘時間

## 系統需求
- Python 3.8 或更高版本
- PyQt5
- requests 庫
- yt-dlp（首次使用時會自動下載）

## 安裝與使用

### 安裝步驟
1. 確保已安裝 Python 3.8 或以上版本
2. 克隆或下載本專案
3. 安裝依賴：
   ```shell
   pip install -r requirements.txt
   ```
4. 運行程式：
   ```shell
   python main.py
   ```

### 使用說明

#### 簡單模式
1. 貼上 YouTube 或其他視頻網站的連結
2. 選擇下載格式（MP4、MP3、M4A）
3. 點擊「下載影片」開始下載

#### 進階模式
1. 貼上視頻連結
2. 點擊「取得格式清單」
3. 選擇喜歡的影片格式和音頻格式
4. 可選：勾選「合併成單個影音檔」選項
5. 點擊「下載影片」開始下載

#### 片段下載
1. 選擇「指定時間下載」
2. 輸入開始和結束時間（格式：HH:MM:SS）
3. 選擇下載模式和格式
4. 點擊「下載影片」

## 故障排除

### 出現下載錯誤
- 首先點擊「自動更新 yt-dlp」更新下載器
- 確保網路連線正常
- 檢查視頻連結是否有效

### 格式無法選擇
- 重新點擊「取得格式清單」
- 確保輸入的是正確的視頻連結
